### Introduction
This approach provides a simple solution for auto-sign-in in NTU's signing website and provide a random time sign-in function. Simple libraries such as `selenium`, `apscheduler` and `random` were used to realize this thought.

### Environment
`Python >= 3.7` (confirmed)

### Usage
Edit the account name and password for user:
```{python}
username: "your_username"
password: "your_password"
```

Execute the program code and then confirm that the computer will not shut down.
